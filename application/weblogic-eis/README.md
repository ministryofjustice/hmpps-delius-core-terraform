# WebLogic - Delius External Interfacing Systems (EIS) domain

Supports the application interfaces for National Delius including connectivity to NOMIS, OFFLOC/DSS and IAPS.

Defines a load-balanced ECS service, running the NDelius application on WebLogic 14c.
